HTML

Cheat Sheet And Guide

- Basic Structure -

<!-- This is a comment in HTML -->
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document Title</title>
</head>
<body>
<h1>Welcome to My Webpage</h1>
<p>This is a simple HTML document.</p>
</body>
</html>

- Common Tags -

Tag Description Example
<html> Root element of the page <html></html>
<head> Contains metadata (title, links, scripts) <head></head>
<meta> Provides metadata like charset and viewport <meta charset="UTF-8">
<title> Sets the document title (visible in browser tab) <title>Page Title</title>
<body> Contains the visible page content <body></body>
<h1> - <h6> Headings, <h1> is the largest <h6> is the smallest <h1>Main Title</h1>
<p> Paragraph text <p>This is a paragraph.</p>
<a> Hyperlink (anchor tag) <a href="https://example.com"> Visit Example</a>
<img> Embeds an image <img src="image.jpg" alt="Description">
<li> List item <li>Item</li>
<ul> Unordered list <ul><li>Item 1</li></ul>
<ol> Ordered list <ol><li>Item 1</li></ol>
<td> Table cell <td>Content</td>
<tr> Table row <tr><td>Content</td></tr>
<table> Table container <table><tr><td>Content</td></tr></table>

- Text Formatting Tags -

Tag Description Example
<strong> Bold text (emphasized) Important
<em> Italicized text (emphasized) <em>Emphasized</em>
<br> Line break (no closing tag) Line 1<br>Line 2
<u> Underlined text <u>Underlined</u>
<mark> Highlighted text <mark>Highlighted</mark>
<sub> Subscript H<sub>2</sub>O
<sup> Superscript x<sup>2</sup>

- Form Elements -

Tag Description Example
<form> Defines a form for input <form action="/submit"></form>
<input> Input field <input type="text" name="username">
<textarea> Multiline text input <textarea name="message"></textarea>
<button> Clickable button <button type="submit"> Submit</button>
<label> Labels an input element <label for="username"> Username:</label>
<option> Defines options in a dropdown <option value="1">Option 1</option>
<select> Dropdown menu <select><option value="1">Option 1</option></select>
<legend> Caption for the <fieldset> <legend>Personal Info</legend>
<fieldset> Groups related form elements <fieldset><legend>Details</legend> ...</fieldset>

- Multimedia -

Tag Description Example
<audio> Embeds an audio file <audio controls><source src="audio.mp3" type="audio/mp3"></audio>
<video> Embeds a video file <video controls><source src="video.mp4" type="video/mp4"></video>
<source> Specifies media resources <source src="movie.mp4" type="video/mp4">
<iframe> Embeds another webpage < src="https://example.com"></iframe>

- Links & Navigation -

Tag Description Example
<a> Hyperlink <a href= "https://example.com">Go to Example</a>
<nav> Defines navigation links <nav><a href="#home">Home</a></nav>
<link> Links external resources (like CSS) <link rel="stylesheet" href="style.css">
<meta> Provides metadata about the document <meta name="description" content="Webpage description">

- Layout & Structure -

Tag Description Example
<div> Generic container for block-level elements <div class="container"> </div>
<span> Inline container for text or elements <span class="highlight"> Text</span>
<header> Defines the header of a section <header>Header content</header>
<footer> Defines the footer of a section <footer>Footer content</footer>
<section> Defines a section in the document <section> Content</section>
<article> Defines independent content <article>Article content</article>
<aside> Defines content aside from the main content <aside>Sidebar content</aside>

- Attributes -

Attribute Description Example
id Unique identifier for an element <div id="unique"></div>
class Defines a class for styling <div class="myClass"></div>
style Inline CSS styles <p style="color: red;">Text</p>
href URL for a link <a href= "https://example.com"> Link</a>
src Source for an image or media <img src="image.jpg" alt="Description">
alt Alternate text for images <img src="image.jpg" alt="Description">
target Specifies where to open a link <a href= "https://example.com" target="_blank">New Tab</a>
placeholder Placeholder text for input fields <input type="text" placeholder="Enter your name">

- HTML Entities -

Entity Description Example
&lt; Less than symbol (<) <div>
&gt; Greater than symbol (>) >div&lt;
&amp; Ampersand (&) &copy;
&copy; Copyright symbol (©) &copy; 2024
