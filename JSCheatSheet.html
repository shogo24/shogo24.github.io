JavaScript

Cheat Sheet And Guide

- Basic Structure -

// This is a comment in JavaScript
// Declaring a variable
let message = "Hello, World!";

// Function definition
function greet() {
console.log(message);
}

// Calling the function
greet();

- Variables & Data Types -

Property Description Example
var Declares a variable with function scope. var x = 10;
let Declares a block-scoped variable. let y = 20;
const Declares a constant (block-scoped). const z = 30;
Number Represents numerical values (integers or floating point). let num = 42;
String Represents a sequence of characters (text). let str = "Hello World";
Boolean Represents true or false values. let isActive = true;
Object Represents a collection of key-value pairs. let person = { name: "John", age: 30 };
Array Represents a collection of items in an ordered list. let arr = [1, 2, 3, 4];
undefined Represents an uninitialized variable. let x; // undefined
null Represents a null or empty value. let empty = null;

- Operators -

Operator Description Example
+ Addition or string concatenation. let sum = 5 + 3;
let greet = "Hello" + " World";
- Subtraction. let diff = 10 - 5;
* Multiplication. let product = 4 * 2;
/ Division. let quotient = 10 / 2;
% Modulus (remainder of division). let remainder = 10 % 3;
++ Increment (increase by 1). let i = 5; i++;
-- Decrement (decrease by 1). let i = 5; i--;
= Assignment (assign a value). let x = 10;
== Equality (checks if values are equal). 5 == '5' // true
=== Strict equality (checks if values and types are equal). 5 === '5' // false
!= Inequality (checks if values are not equal). 5 != 4 // true
!== Strict inequality (checks if values and types are not equal). 5 !== '5' // true
> Greater than. 5 > 3 // true
< Less than. 5 < 3 // false
>= Greater than or equal to. 5 >= 5 // true
<= Less than or equal to. 5 <= 10 // true

- Control Flow -

Keyword Description Example
if Executes code if condition is true. if (x > 10) { console.log("Greater"); }
else Executes code if condition is false. if (x > 10) { console.log("Greater"); } else { console.log("Smaller"); }
else if Checks multiple conditions. if (x > 10) { console.log("Greater"); } else if (x == 10) { console.log("Equal"); }
switch Selects one of many code blocks to execute based on a variable's value. switch(day) { case 1: console.log("Monday"); break; }
for A loop that runs for a specific number of times. for (let i = 0; i < 5; i++) { console.log(i); }
while A loop that runs while a condition is true. while (x < 5) { console.log(x); x++; }
do...while A loop that runs at least once before checking the condition. do { console.log(x); x++; } while (x < 5);

- Functions -

Property Description Example
Function Declaration Declares a function. function greet() { console.log("Hello!"); }
Function Expression Defines a function within an expression. const greet = function() { console.log("Hello!"); };
Arrow Function Shorter syntax for defining functions. const greet = () => { console.log("Hello!"); };
Return Returns a value from a function. function add(a, b) { return a + b; }
Parameters Values passed to a function. function greet(name) { console.log("Hello " + name); }
Arguments Values supplied to a function when called. greet("John");

- Objects & Arrays -

Property Description Example
Object Literal Defines an object with properties and values. let person = { name: "John", age: 30 };
Array Literal Defines an array with values. let arr = [1, 2, 3, 4];
Accessing Object Property Access an object property using dot notation or bracket notation. person.name or person["name"]
Accessing Array Element Access an element in an array by index. arr[0] // 1
Array Methods Common methods like push(), pop(), shift(), unshift(). arr.push(5);
Object Methods Define methods within objects. let person = { name: "John", greet() { console.log("Hello"); } };

- Events -

Event Description Example
addEventListener() Attaches an event handler to an element. button. addEventListener ("click", function() { alert("Clicked!"); });
onclick Triggered when an element is clicked. button.onclick = function() { alert("Clicked!"); };
onmouseover Triggered when mouse hovers over an element. div.onmouseover = function() { console.log ("Mouse over!"); };
onchange Triggered when the value of an input element changes. input.onchange = function() { console.log ("Changed!"); };

- ES6+ Features -

Feature Description Example
Template Literals Allows embedding expressions inside strings using backticks. let name = "John"; let greeting = `Hello, ${name}!`;
Destructuring Assignment Extracts values from arrays or objects and assigns them to variables. let [a, b] = [1, 2]; let {name, age} = person;
Spread Operator Expands elements of an array or object. let arr = [1, 2, 3]; let newArr = [...arr, 4];
Rest Parameter Collects arguments into an array in function definitions. function sum(...numbers) { return numbers.reduce((a, b) => a + b); }
Promises Represents asynchronous operations. let promise = new Promise((resolve, reject) => { resolve("Done"); });

- Asynchronous Programming -

Method Description Example
setTimeout() Calls a function after a specified time. setTimeout(() => { console.log("Hello after 2 seconds!"); }, 2000);
setInterval() Repeatedly calls a function after a specified interval. setInterval(() => { console.log("Hello every 2 seconds!"); }, 2000);
async/await Allows working with promises in a synchronous style. async function fetchData() { let response = await fetch(url); }
