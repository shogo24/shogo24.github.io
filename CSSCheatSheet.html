CSS

Cheat Sheet And Guide

- Basic Structure -

/* This is a comment in CSS */
/* Global styles */
body {
font-family: Arial, sans-serif;
background-color: #f4f4f4;
margin: 0;
padding: 0;
}

h1 {
color: #333;
text-align: center;
}

p {
color: #666;
font-size: 16px;
line-height: 1.5;
}

- Text Properties -

Property Description Example
color Sets the color of text. color: red;
font-family Defines the font to be used for text. font-family: 'Arial', sans-serif;
font-size Sets the size of the font. font-size: 16px;
font-weight Sets the thickness of the font. font-weight: bold;
line-height Sets the height between lines of text. line-height: 1.5;
text-align Aligns the text inside an element. text-align: center;
text-decoration Adds decoration to text (underline, overline, etc.). text-decoration: underline;
text-transform Controls the capitalization of text (uppercase, lowercase, capitalize). text-transform: uppercase;
letter-spacing Controls the space between characters. letter-spacing: 1px;
word-spacing Controls the space between words. word-spacing: 5px;

- Box Model & Spacing -

Property Description Example
margin Sets the space outside an element (outside the border). margin: 10px;
padding Sets the space inside an element, between the content and the border. padding: 10px;
border Defines the border around an element. border: 2px solid black;
width Sets the width of an element. width: 100px;
height Sets the height of an element. height: 200px;
box-sizing Defines how the total width and height are calculated (including padding and borders). box-sizing: border-box;
display Defines how an element is displayed (e.g., block, inline, flex, grid). display: block;
overflow Specifies what happens if content overflows an elementâ€™s box (scroll, hidden, etc.). overflow: auto;

- Positioning -

Property Description Example
position Specifies how an element is positioned (static, relative, absolute, fixed, sticky). position: relative;
top, right, bottom, left Offsets the position of an element (used with relative or absolute positioning). top: 20px; left: 30px;
z-index Specifies the stack order of an element (higher numbers appear on top). z-index: 10;
float Allows an element to float to the left or right, affecting text and other elements around it. float: left;
clear Clears floated elements, affecting the layout of the next element. clear: both;

- Flexbox -

Property Description Example
display: flex Defines a flex container, enabling flexbox layout. display: flex;
flex-direction Specifies the direction of flex items (row, column). flex-direction: row;
justify-content Aligns flex items along the main axis (e.g., left, center, right). justify-content: center;
align-items Aligns flex items along the cross axis (e.g., top, center, bottom). align-items: stretch;
flex-wrap Defines whether flex items should wrap or not. flex-wrap: wrap;
flex A shorthand for flex-grow, flex-shrink, and flex-basis. flex: 1 1 0%;

- Grid Layout -

Property Description Example
display: grid Defines a grid container. display: grid;
grid-template-columns Defines the number and size of columns in a grid. grid-template-columns: 100px 1fr 2fr;
grid-template-rows Defines the number and size of rows in a grid. grid-template-rows: 50px 200px;
grid-gap Sets the gap between rows and columns in a grid. grid-gap: 10px;
grid-column Specifies how many columns an item should span. grid-column: span 2;
grid-row Specifies how many rows an item should span. grid-row: span 2;

- Styling & Effects -

Property Description Example
border-radius Rounds the corners of an element. border-radius: 10px;
box-shadow Adds shadow effects to elements. box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
text-shadow Adds shadow to text. text-shadow: 1px 1px 2px #000000;
opacity Sets the transparency level of an element. opacity: 0.5;
transform Allows you to rotate, scale, skew, or move an element. transform: rotate(45deg);
transition Defines the transition effect when a property changes (e.g., fade, slide). transition: all 0.3s ease;
animation Defines animations for elements (keyframes, durations). animation: fadeIn 2s ease-in-out;

- Miscellaneous -

Property Description Example
visibility Specifies whether an element is visible or hidden (without removing it from the document flow). visibility: hidden;
cursor Changes the mouse cursor when it hovers over an element. cursor: pointer;
filter Applies visual effects like blur or brightness to elements. filter: blur(5px);
content Specifies content to insert in pseudo-elements (e.g., ::before, ::after). content: "New";
